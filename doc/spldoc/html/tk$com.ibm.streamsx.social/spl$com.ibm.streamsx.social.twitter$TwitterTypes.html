Types

UserType

This type matches the user JSON attribute in a tweet. See https://dev.twitter.com/docs/platform-objects/users

UserType = boolean verified, rstring name, rstring id_str, int64 listed_count, int64 utc_offset, int64 id, rstring time_zone, int64 friends_count, int64 followers_count, rstring created_at, boolean default_profile, rstring lang, int64 statuses_count, rstring location, boolean contributors_enabled, int64 favourites_count, rstring description, rstring screen_name;

CoordinatesType

Coordinates of the user

CoordinatesType = list<float64> coordinates;

TweetType

This type matches the names of the JSON attributes in a tweet. See https://dev.twitter.com/docs/platform-objects/tweets for details

TweetType = CoordinatesType coordinates, UserType user, boolean favorited, rstring created_at, rstring text, rstring id_str, int64 retweet_count, int64 id, rstring source, boolean retweeted;
